# Titanic Survival Prediction using Machine Learning

## Overview

This project focuses on predicting passenger survival on the Titanic dataset using multiple machine learning algorithms.

## Models Used

* K-Nearest Neighbors (KNN)
* Logistic Regression
* Decision Tree Classifier

## Dataset

The dataset contains passenger information such as age, gender, ticket class, and fare.

## Steps Performed

* Data cleaning (handling missing values)
* Feature engineering and encoding
* Data scaling using StandardScaler
* Model training and prediction
* Performance comparison

## Results

* Logistic Regression achieved the highest accuracy (~94%)
* KNN achieved moderate accuracy (~81%)
* Decision Tree showed slightly lower performance (~79%)

## Conclusion

Logistic Regression performed the best for this dataset, showing that linear models can be highly effective for classification problems.
